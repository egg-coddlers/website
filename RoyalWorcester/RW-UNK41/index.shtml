Royal Worcester - RW-UNK41 (Blue Oriental Motif)

This pattern consists of a blue oriental motif that has some feature similarities to the Willow pattern, but it isn't the same. According to a contact at Royal Worcester, a version of the Willow pattern was produced between 1890 and 1902 at the Grainger factory. We don't know if this is it.
