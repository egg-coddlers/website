Royal Worcester - Easter Bunnies Pattern

This is a newly discovered Royal Worcester pattern. The one example that we've heard about seems to be several years old. The pattern consists of an Easter motif (bunny, chicks, basket, decorated eggs, chocolate bunnies, etc). Another Royal Worcester pattern with a similar motif (possibly part of the same pattern) is RW-UNK44.
