Royal Worcester - Pattern Name Unknown (Spode Christmas Tree)

This pattern consists of the famous Christmas Tree pattern by Spode Ltd. The box in which this coddler is sold is labelled with the Spode Logo, and Spode Limited, Church Street, Stoke-on-Trent ST4 1BX, England and a telephone and fax number.

I suspect that Spode has/had RW make up these coddlers for their Christmas Tree line of giftware. They are not listed in the online Spode.
