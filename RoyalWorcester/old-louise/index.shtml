Royal Worcester - (Old) Louise Pattern

The pattern named Louise was first produced by Royal Worcester around the 1930's. Later, it was replaced by a similar pattern that incorporated many of the same elements, but on slightly larger scale. The new pattern was also called Louise.

This pattern consists of hand-painted multi-coloured flowers. There is considerable variation among the combinations of flowers that were used on each example.

We will refer to the older pattern as (Old) Louise, and the newer pattern as Louise.

We are interested in examining as many different examples of this pattern as we can. If you have a coddler (or coddlers) in your collection that seem similar to this pattern, please do contact us so that we can acquire images of the pattern and the backstamp.
