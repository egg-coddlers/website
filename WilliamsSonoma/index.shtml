Egg Coddlers from Williams Sonoma

There are several egg coddlers that are cylindrical in shape, all were sold by Williams-Sonoma at some point in time.

The earlier versions of these coddlers have round stickers on the bottom that look like backstamps. The more recent ones have a backstamp.
