Maling Egg Coddlers

These are single sized coddlers. We have also heard rumours of a double-sized coddler.

Very little is know about Maling coddlers, other than this one pattern. All that we have seen are very similar, but each is slightly different. We assume them to be one pattern - probably either hand painted, or selected from a common set of decals.

C.T. Maling was a Newcastle-on-Tyne based pottery.
