Coalport - Willow

This is the classic Willow pattern. The backstamp includes the name of the pattern. Pictured is a double-sized coddler. A single-size version also exists, which may be considerably older than the double version. We are not sure whether the official name of the single size is also Willow, so we refer to it as CO-UNK08.
