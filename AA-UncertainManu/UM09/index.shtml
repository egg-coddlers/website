Unknown Manufacturer - UM09

These coddlers are double-sized coddlers with labels inscribed Made in Japan and a number on the bottom. An instruction sheet that accompanies one of these coddlers indicated the name Butterfly Garden for the pattern. This pattern seems to be the same as a pattern by Royal Crown of Japan, called "Spring Time."

These coddlers are found with the same pattern in both a convex style, with a pedestal, and a concave style, with no pedestal. This suggests that they might have been manufactured by Shafford.

Some examples of this coddler have a removable sticker on the bottom, with the words: HAND PAINTED, LENWILLE CHINA, ARDALT JAPAN

BB has seen other pieces, such as plates and dishes, with the same designs but different numbers underneath. This fact would imply that the numbers are the manufacturer's part number. However, there isn't consistency in the examples we've seen. We have seen examples of the convex with pedestal coddler, both with the numbers 7051 and 7215 on the bottom. We've seen examples of the concave coddler with the number 7051 only, so far.

The numbers on the bottom are inked in either red or dark blue.
